# takeover Scene Sprites

* Render takeover all sprites in current scene

```sig
Render.takeoverSceneSprites()
```

## ~hint
* Render will call this automatically, so not required usually.
* Call this just after created the sprite, only if you saw sprite draw at its tilemap position on screen
## ~

## Parameters
* none

## Returns

* none
